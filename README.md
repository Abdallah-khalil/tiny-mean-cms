### Prerequisites
* [NodeJS](https://nodejs.org/en/)
* [MongoDB](https://www.mongodb.com/)
* [Brunch](http://brunch.io/)
* [Bower](https://bower.io/)

### Installation

```bash
git clone https://github.com/jbehuet/mean-starter-es6.git   
cd mean-starter-es6
npm i
bower i
```

### Development

`npm run dev`

### Production

`npm start`
